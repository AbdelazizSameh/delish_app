# DelishApp

DelishApp is a mobile application built using **Flutter** that allows users to browse restaurants, explore meals, add items to the cart, and place orders easily. The app supports **two roles**: User and Admin.

----------

## ğŸ“Œ Overview

DelishApp provides a smooth user experience for searching restaurants and meals, adding items to the cart, tracking orders, and managing the user account.  
It is developed using **Flutter & Firebase**, with state management via **Bloc** and following **Clean Architecture** principles.
